## Introduction

This chart bootstraps a [ProxyWeb](https://github.com/edmodo/proxyweb/) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.

## Prerequisites

- Kubernetes 1.19+
